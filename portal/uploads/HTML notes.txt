HTML stand for: Hyper Text Markup Language

Vocab:

Semantics: Describing meaning for data
HTML5: Newest version of the spec
Tags: Create the elements *displays in source-code*
Element: Things that end up rendered
DOM: Document outline model, outline of elements rendered on screen
Node: Items shown in the DOM
Nesting Elements: Wrapping content with a tag creates meaning whereas wrapping tags provides organization.
